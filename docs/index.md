# 통신사 구독 연동 문서 묶음

이 문서 묶음은 통신사 멤버십/부가서비스를 통한 **OGQ 구독 연동** 프로젝트의
설계, 후보 비교, 의사결정, 파트너 문의 내역을 정리한 것입니다.

## 문서 목록

| 문서 | 내용 | 상태 |
| --- | --- | --- |
| [01. 설계](01_design.html) | 연동 아키텍처 및 흐름도 | 검토중 |
| [02. 후보 비교](02_candidates.html) | 연동 방식 후보 비교표 | 확정 |
| [03. 결정 게이트](03_decision_gate.html) | 최종 의사결정 기록 | 검토중 |
| [04. 파트너 문의서](04_partner_inquiry.html) | 통신사 파트너 발송용 문의서 | 초안 |

## 읽는 순서

1. 먼저 [설계 문서](01_design.html)에서 전체 그림을 파악합니다.
2. [후보 비교](02_candidates.html)에서 검토한 대안들을 확인합니다.
3. [결정 게이트](03_decision_gate.html)에서 최종 채택안과 근거를 확인합니다.
4. 외부 파트너와의 커뮤니케이션은 [파트너 문의서](04_partner_inquiry.html)를 참고합니다.

> 본 문서는 내부 검토용이며, 외부 공유 전 반드시 결정 게이트 승인 여부를 확인하세요.
