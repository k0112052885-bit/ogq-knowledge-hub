          환경설정

          서버를 종료할까요?

          로컬 편집 서버(npm run dev)가 종료됩니다. 저장하지 않은 변경사항이 있다면 먼저 저장 버튼을 눌러주세요.

          Git Push

          git add -A → git commit → git push 를 순서대로 실행합니다. main 브랜치에 push되면 GitHub Actions가 자동으로 빌드하고 GitHub Pages에 배포합니다.
